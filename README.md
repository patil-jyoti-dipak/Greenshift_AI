 GreenShift AI — Renewable Energy Adoption Predictor
Overview
GreenShift AI is a competition-grade, end-to-end machine learning application that predicts whether a country or region will adopt renewable energy such as Solar, Wind, or Hydro based on socio-economic and environmental factors. The project is built using Logistic Regression and covers the complete ML lifecycle from data collection and cleaning to model deployment via an interactive Streamlit web application.
Problem Statement
Every year, governments and energy companies waste billions of dollars targeting the wrong markets for renewable energy campaigns. Without a smart, data-driven prediction system, subsidies reach the wrong regions, awareness campaigns fail to convert, and global climate goals are missed. GreenShift AI addresses this gap by using machine learning to predict adoption likelihood based on real-world economic, environmental, and social indicators.
Dataset
The dataset is sourced from Kaggle, published by Tarunesh Burman, and contains 1,000 records with 11 features covering the period 2000 to 2022. The target variable is Adoption_Status which is divided into three classes — Class 0 represents No Adoption with 318 records, Class 1 represents Partial Adoption with 365 records, and Class 2 represents Full Adoption with 317 records. The key features include Solar Exposure, Wind Speed, Household Income, Government Subsidy, Electricity Price, Awareness Programs, Urbanization Rate, Population Density, Grid Accessibility, and Region.
ML Pipeline
The project follows a structured 7-step pipeline. First, the dataset is collected from Kaggle. Second, data cleaning is performed including duplicate removal, outlier capping using the 3×IQR method, and label encoding of categorical columns. Third, a thorough Exploratory Data Analysis is conducted covering Univariate, Bivariate, and Multivariate analysis including correlation heatmaps, VIF analysis, and mutual information scores. Fourth, preprocessing is applied using RobustScaler for feature scaling, SMOTE for handling class imbalance, and Recursive Feature Elimination to select the top 8 features. Fifth, six machine learning classifiers are trained and compared using 5-fold stratified cross-validation. Sixth, GridSearchCV is used for hyperparameter tuning of the final Logistic Regression model. Finally, the model is deployed as an interactive Streamlit web application called GreenShift AI.
Algorithm
The primary algorithm used is Logistic Regression configured for Multinomial multiclass classification. The model uses the SAGA solver, L2 Ridge regularization, balanced class weights, and a maximum of 5,000 iterations. The mathematical formula is P = 1 / (1 + e⁻ᶻ) where z = w₀ + w₁x₁ + w₂x₂ + ... + wₙxₙ and the class with the highest probability is assigned as the predicted label. Logistic Regression was chosen for its interpretability, ability to output class probabilities, fast training speed, and native support for multiclass problems.
Model Results
Six models were compared. Logistic Regression achieved an AUC of 0.523 and F1 of 0.349. KNN achieved the best AUC of 0.548 and F1 of 0.381. Decision Tree achieved AUC 0.541 and F1 0.379. Random Forest achieved AUC 0.528 and F1 0.337. Gradient Boosting achieved AUC 0.519 and F1 0.326. Naive Bayes achieved AUC 0.501 and F1 0.298. The optimised Logistic Regression model achieved a final accuracy of 36%, balanced accuracy of 36.7%, weighted F1 score of 34.9%, and ROC-AUC of 0.523 on the test set.
Web Application
GreenShift AI is deployed as a Streamlit web application with 7 interactive pages. The Dashboard page shows KPIs, adoption trends, and energy mix charts. The Predict page allows real-time adoption prediction with a live gauge chart and log-odds breakdown. The EDA Explorer provides interactive Univariate, Bivariate, and Multivariate analysis. The Global Map displays a country-level choropleth map. The Model Insights page shows the ROC curve, confusion matrix, and feature importance. The Batch Analysis page allows users to upload a CSV file for bulk predictions and download results. The About page provides full project and tech stack information.
Tech Stack
The project is built entirely in Python 3.12 using Scikit-learn for the ML pipeline, Streamlit for the web application, Plotly for interactive visualizations, Pandas and NumPy for data processing, Imbalanced-learn for SMOTE resampling, Statsmodels for VIF analysis, Matplotlib and Seaborn for EDA plots, and Joblib for model saving and loading.
How to Run
Clone the repository, install the dependencies using pip install -r requirements.txt, and launch the app using streamlit run app.py.
